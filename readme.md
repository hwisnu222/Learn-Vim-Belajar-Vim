# Belajar VIM

## Mode dalam vim
NORMAL =    ESC 
INSERT = i
VISUAL = v

## Copy paste code

`
Mode visual

copy = y

cut = d

paste = p

untuk mempaste dibawah kode mengukan p(kecil). untuk sebelum kode menggunakan P(besar)
`
